Contributing
============

Thanks for contributing to NeoTalon! Please follow these guidelines to keep the codebase high-quality.

Formatting
- We use Stylua for formatting. Install it and run:

  stylua .

Linting
- We use luacheck for linting. Install it and run:

  luacheck .

If you don't have these tools installed, please leave the code consistent with the project's style and add sensible changes in a separate commit.

Making changes
- Keep functions small and avoid polluting the global namespace.
- Add tests for new behaviour when possible (not yet available for this repo).
- Update CHANGELOG.md with a short description of your change.
